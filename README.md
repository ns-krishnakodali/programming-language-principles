# Programming Language Principles

This repository contains my work for the COP-5556 course on Programming Language Principles.

The course focuses on understanding how programming languages are designed, specified, interpreted, and analyzed, with a primary emphasis on practical implementations.

---

## Overview

I explore the core principles behind language design and implementation, including:

- Syntax and formal grammars
- Parsing techniques
- Abstract syntax trees
- Static and dynamic semantics
- Type systems
- Scope and binding
- Runtime environments
- Memory models
- Intermediate representations

The emphasis is on building foundational knowledge in how high-level languages are translated and executed.

For each project in this repository, I included a dedicated `README` file with instructions for setup and execution. Please refer to the respective project `README` for details.

---

## Language Implementation

As part of this work, I explore the implementation of:

- Parse tree construction
- AST traversal using visitor patterns
- Semantic analysis
- Runtime evaluation
- Code generation fundamentals

---

## Technical Focus

Key technical areas include:

- Context-free grammars and language recognition
- LL and LR parsing strategies
- Recursive descent parsing
- Static semantic checks
- Symbol table management
- Control flow representation
- Type checking and inference concepts
- Execution models and environment management

This work connects formal language theory with practical systems-level implementation.

The objective of this repository is to document and demonstrate my understanding of programming language construction, from grammar specification through execution semantics.
